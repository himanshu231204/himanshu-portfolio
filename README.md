## üõ†Ô∏è Run Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/himanshu231204/himanshu-portfolio.git
````

2. Create virtual environment and activate:

   ```bash
   python -m venv venv
   ```


